# E-Commerce Platform

Интернет-магазин с личным кабинетом и валидацией, корзиной покупок и избранным. Full-stack приложение на React и Node.js.

## Функционал

- **Регистрация и вход** — JWT аутентификация с защищенными маршрутами
- **Личный кабинет** — просмотр и редактирование профиля
- **Каталог товаров** — просмотр товаров из базы данных
- **Корзина** — добавление/удаление товаров, изменение количества
- **Избранное** — сохранение понравившихся товаров
- **Заказы** — оформление и история заказов

## Технологии

### Frontend
- **React 18** — библиотека для интерфейсов
- **Tailwind CSS** — стилизация компонентов
- **React Router DOM v6** — навигация между страницами
- **React Query** — управление состоянием и кеширование
- **Axios** — HTTP запросы к API

### Backend
- **Node.js** — среда выполнения JavaScript
- **Express** — веб-фреймворк для API
- **PostgreSQL** — база данных
- **Sequelize ORM** — работа с базой данных через модели
- **bcryptjs** — хеширование паролей
- **jsonwebtoken** — JWT токены для аутентификации
